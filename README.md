# Travel Splitter

엑셀 파일과 카드사 이용내역 복사 텍스트에서 정산 후보를 읽고, KRW/JPY/USD/THB/VND/CNY/EUR가 섞인 지출을 한 기준 통화로 환산해 1/n 정산 링크를 만드는 정적 웹앱입니다.

## 가져오기

- `.xlsx`, `.csv`, `.tsv` 파일 업로드를 지원합니다.
- `.xlsx`는 브라우저에서 직접 읽습니다. GitHub Pages나 Netlify 같은 정적 배포 환경에서도 파일 업로드가 동작합니다.
- 엑셀에서 그대로 복사한 탭 구분 텍스트도 붙여넣을 수 있습니다.
- KB카드 앱/웹 이용내역처럼 `06월29일`, `가맹점`, `금액`, `시간`이 반복되는 복사 텍스트도 붙여넣을 수 있습니다.
- 신한카드 이용내역처럼 `가맹점`, `금액`, `2026.06.29 10:59 ...`가 반복되는 복사 텍스트도 붙여넣을 수 있습니다.
- 지원 헤더: `날짜`, `자산`, `분류`, `소분류`, `내용`, `금액(원)`, `수입/지출`, `메모`, `금액`, `화폐`
- `분류`가 `여행 / 공금`이고 `수입/지출`이 `지출`인 행을 정산 후보로 가져옵니다.
- 외화 행은 `금액`과 `화폐`를 원금으로 저장하고, `금액(원)`을 카드 청구 원화 환산액으로 사용합니다.

## 결제 내역 관리

- 내역 탭에서 결제일, 금액, 내용 기준으로 정렬할 수 있습니다.
- 결제일 정렬은 엑셀의 시간 정보가 있으면 같은 날짜 안에서도 시간순으로 정렬합니다.
- 각 행은 날짜, 결제 내용, 결제자, 분할 인원, 원금 통화와 정산 기준 금액을 함께 보여줍니다.
- 여러 결제를 선택한 뒤 태그, 결제자, 전체 인원 분할, 한 사람 부담, 직접 비율을 한 번에 적용할 수 있습니다.
- 결제 행의 `빼기`/`넣기` 버튼으로 개인 사용 내역을 정산 총액에서 제외하거나 다시 포함할 수 있습니다.
- 결제 제목 영역을 누르면 결제 내용, 태그, 결제일, 결제자, 원금 통화 금액, 원화 환산액, 사람별 부담 비율을 직접 수정할 수 있습니다.
- 부담 비율을 조정하면 1명 부담, 몰아주기, 가중 분담 정산을 만들 수 있습니다.
- 받을 금액과 보낼 금액은 현실 송금에 맞춰 `10원` 단위로 반올림해 보여줍니다.
- 정산 목록과 정산표에는 입력된 결제 내역의 첫 날짜와 마지막 날짜를 기준으로 `3월 3일 - 3월 11일`처럼 표시됩니다.

## 통화

- 각 결제 내역은 `KRW`, `JPY`, `USD`, `THB`, `VND`, `CNY`, `EUR` 중 하나의 원금 통화를 가집니다.
- 가져오기 탭의 환율 입력에서 외화별 `KRW` 환산값을 바꾸면 전체 정산 결과가 다시 계산됩니다.
- 정산 기준 통화를 바꾸면 받을 돈과 보낼 돈이 해당 통화로 표시됩니다.

## 정산 관리

- 정산 선택 드롭다운에서 과거/현재/미래 일정을 전환합니다.
- `새 여행`으로 빈 정산표를 만들고, `복제`로 참가자와 설정을 이어받은 복사본을 만들 수 있습니다.
- `정산 삭제`는 현재 정산을 목록에서 제거합니다.
- `현재 내역 전체 삭제`는 선택한 정산의 결제 내역만 모두 지웁니다.
- 시작일과 종료일은 결제일 범위와 별도로 직접 수정할 수 있습니다.
- 기록은 브라우저에 자동 저장됩니다. 기기를 바꾸거나 브라우저 저장소가 지워질 때를 대비해 정산 탭에서 `기록 파일 저장`과 `기록 불러오기`를 사용할 수 있습니다.
- GitHub Pages 주소만 보내면 빈 앱이 열립니다. 현재 선택한 정산 데이터까지 같이 보내려면 정산 탭의 `정산 링크 만들기`로 만들어진 긴 링크를 공유합니다.

## 정산표 공유

- 정산 탭에서 사람별 결제액, 부담액, 받을 돈/보낼 돈을 한 번에 확인할 수 있습니다.
- 보내실 곳 계좌는 정산 탭에서 직접 입력한 경우에만 정산 화면과 저장 이미지에 표시됩니다.
- `계좌번호 복사하기`로 계좌 정보를 복사하고, `전체 이미지 저장`으로 정산표와 결제 내역이 이어진 긴 PNG 파일을 만들 수 있습니다.
